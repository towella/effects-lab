# blob-procedural-animation-test
Test proof-of-concept game made with pygame

By Andrew Towell (2022 - 23)

## Main File
- main.py

## Required
- python 3.7
- pygame 2.0.1
- pyinstaller 5.1
- generate-iconset 1.2.0

## Controls
| Key(s)     | Function       |
|------------|----------------|
| Move Mouse | Move Orb       |
| G          | Toggle Gravity |
| M          | Decrease Tail  |
| N          | Increase Tail  |
| ',' or ESC | Quit           |
| P          | Pause          |
